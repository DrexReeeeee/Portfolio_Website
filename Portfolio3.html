NUMBER SYSTEMS: TYPES, USES, AND SIGNIFICANCE

Explore number systems, their types, uses, and significance in computer science and digital technology.

What are Number Systems?

> Number systems are foundational concepts in mathematics, providing a framework for representing and manipulating numerical quantities (Floyd, 2015).

> They are useful in digital computers and the knowledge of these systems is necessary to perform reliable, economic and easily under stable arithmetic operations.

> Number systems are also known as positional number systems since the value of each symbol (i.e., digit and alphabet) in a number is determined by its position within the number (NCERT, 2024).

Structure of Number Systems

Place Value: Critical for interpreting digits in various bases.

Properties: Includes number types (integers, fractions) and properties like commutativity (Lay et al., 2016).

Ordering and Comparison: Arranging and assessing numbers in different bases.

Simplifying Expressions: Reducing and solving equations in various bases (Rosen, 2019).

Types of Number Systems

Binary Number System

> The binary number system is represented by only two symbols, 0 and 1 (Lande, 2014).

> Nearly all computers use the binary numeral system which maps directly to the OFF and ON conditions of an electrical switch.

> A binary numeral is a sequence of binary digits, called bits. Each bit position is multiplied by a power of 2 (Weiss, 2006).

> In binary, the column weights ( from right to left) are 1, 2, 4, 8, 16, 32, 64, 128, 256, 512, 1024, 2048, 4096, 8192, and so on. If you work with binary numbers often, you’ll save time if you remember these powers of two up to 2^16 (Buchanan, 1997).

Decimal Number System

> The decimal number system is base-10, using the digits 0 to 9.

> The most widely used number system in daily life, taught from early education.

> Integral to most mathematical computations and real-world applications.

Octal Number System

> The octal number system uses eight symbols: 0 through 7.

> It is often used in computing as a more compact representation of binary numbers. Each octal digit represents three binary digits (bits).

> In octal, the column weights (from right to left) are 1, 8, 64, 512, 4096, and so on. Each digit is multiplied by a power of 8.

> Octal is rarely used, but is studied as an example of how the formalization of the number systems may be applied to any systems regardless of the amount of symbols they contain (LeMeres, 2017).

Hexadecimal Number System

> The hexadecimal number system uses sixteen symbols: 0 through 9 and A through F.

> It is frequently used in computing as a more compact representation of binary numbers.

> Each hexadecimal digit represents four binary bits (Latif & Khan, 2011).

> In hexadecimal, the column weights (from right to left) are 1, 16, 256, 4096, and so on.

> Each digit is multiplied by a power of 16.

Uses/Significance of each number system

Binary Systems

Binary systems are the foundation of digital technology and computing (Palit, 2013).

1. Binary Representation: Data is stored as 0s and 1s, with combinations representing different values.

2. Binary Codes: Unique bit combinations represent discrete information for computer processing.

3. Logic Operations: Computers use binary logic to perform calculations and process signals.

4. Storage and Processing: Information is stored and processed in binary for executing instructions.

Decimal Systems

It is the most familiar and practical number system in daily and professional life (Tanenbaum, 2006).

1. Everyday Calculations: Used in basic arithmetic and financial transactions.

2. Digital Applications: Interfaces with users in calculators and financial systems.

3. Financial Transactions: Used in finance and accounting for its simplicity with fractional parts.

4. Conversion to Binary:Binary conversion aligns easily with decimal through division by 2.

Octal System

In assessing the evolution of number systems, Weibull (2004) states that the octal number system, while not as extensively used today as binary or hexadecimal, remains important due to the following:

1. Historical Context: Octal was used in computing as a shorthand for binary, with each digit representing three binary bits.

2. Unix File Permissions: In Unix systems, file permissions are represented using octal numbers, with each digit showing a set of permissions.

3. Simplified Representation: Octal offers a compact way to represent binary but is less common today, as hexadecimal is more widely used.

Hexadecimal System

1. Memory Addressing and Efficiency: Hexadecimal efficiently represents memory addresses and data in programming, with each digit representing four binary bits (Sakar, n.d.).

2. Color Codes: In web design and graphics, hexadecimal color codes are used to define colors. For example, #FFFFFF represents white.

3. Machine Code and Assembly Language: Hexadecimal simplifies the representation of machine code and assembly language instructions, which are otherwise lengthy in binary format.
